* Cumulative Grammar Definitions

As of §5.1.3 "A Grammar for Lox expressions"
the current grammar definition is:

#+begin_src text
  expression      → literal
                  | unary
                  | binary
                  | grouping ;

  literal         → NUMBER | STRING | "true" | "false" | "nil" ;
  grouping        → "(" expression ")" ;
  unary           → ( "-" | "!" ) expression ;
  binary          → expression operator expression ;
  operator        → "==" | "!=" | "<" | "<=" | ">" | ">="
                  | "+" | "-" | "*" | "/" ;
#+end_src
